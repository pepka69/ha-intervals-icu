# 🎯 Vision – ha-intervals-icu

## Notre objectif

Créer l'intégration Home Assistant la plus complète, la plus simple à utiliser et la mieux documentée pour Intervals.icu.

L'intégration doit permettre à tous les utilisateurs, débutants comme expérimentés, d'exploiter leurs données sportives directement dans Home Assistant.

---

# Les principes du projet

## Simplicité

Une installation doit pouvoir être réalisée en quelques minutes.

L'utilisateur ne doit pas avoir besoin de connaissances en développement.

La documentation doit expliquer chaque étape.

---

## Qualité

Chaque version doit :

- fonctionner
- être testée
- être documentée
- être traduite

Aucune release ne sera publiée sans documentation correspondante.

---

## Intégration Home Assistant

Le projet doit respecter autant que possible les standards Home Assistant :

- Config Flow
- HACS
- Lovelace UI
- Diagnostics
- Translations
- Repairs
- Quality Scale

---

## Expérience utilisateur

L'utilisateur ne doit pas avoir besoin de connaître :

- les API
- JSON
- YAML
- Git

Tout doit pouvoir être configuré graphiquement.

---

## Documentation

La documentation est considérée comme une fonctionnalité du projet.

Chaque nouvelle fonctionnalité devra être documentée avant publication.

---

## International

Le projet est développé en français mais vise une communauté internationale.

Les langues officiellement maintenues sont :

- 🇫🇷 Français
- 🇬🇧 English

---

# Philosophie

Nous privilégions :

✔ une fonctionnalité bien terminée

plutôt que

❌ dix fonctionnalités incomplètes.

---

# Public visé

Cette intégration est destinée :

- aux cyclistes
- aux pratiquants de CrossFit
- aux coureurs
- aux triathlètes
- aux vététistes
- à tous les utilisateurs d'Intervals.icu

---

# Ce que cette intégration n'est pas

Elle ne cherche pas à remplacer Intervals.icu.

Elle cherche à rendre ses données accessibles dans Home Assistant afin de créer :

- des tableaux de bord
- des automatisations
- des notifications
- des statistiques
- des vues personnalisées

---

# Notre ambition

Devenir la référence Home Assistant pour Intervals.icu.
