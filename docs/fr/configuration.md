# Configuration

## Plusieurs athlètes

Ajoutez une entrée d’intégration par compte Intervals.icu.

Chaque entrée crée son propre appareil et ses propres entités.

## Actualisation manuelle

Action disponible :

```text
ha_intervals_icu.refresh
```

Elle force une mise à jour immédiate.

## Capteur Dashboard

Le capteur Dashboard regroupe les données essentielles dans ses attributs :

- résumé ;
- historique ;
- statistiques hebdomadaires ;
- statistiques mensuelles ;
- santé ;
- records ;
- dernière activité ;
- séances prévues.

## Capteurs personnalisés

Les capteurs de santé externes peuvent venir de n’importe quelle intégration Home Assistant :

- balance ;
- Garmin ;
- Withings ;
- Xiaomi ;
- Sleep as Android ;
- capteurs Template.

Le capteur manuel est utilisé en priorité.
