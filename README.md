# Overview

**Intervals.icu for Home Assistant** connects your Intervals.icu account to Home Assistant and exposes your training, recovery and health metrics as native Home Assistant entities.

Create dashboards, automations, notifications and statistics using your real training data.

Whether you practice cycling, running, CrossFit, triathlon, MTB or any other sport supported by Intervals.icu, your data becomes available throughout your Home Assistant installation.

# Why this integration?

Intervals.icu is one of the best training analysis platforms available.

Home Assistant is one of the best home automation platforms available.

This integration combines both worlds.

Examples:

- Display today's training directly on your dashboard.
- Receive a notification after synchronization.
- Show your current Fitness, Fatigue and Form.
- Display your latest activity on a wall tablet.
- Build automations using your recovery status.
- Monitor your health alongside your home.

# Installation

## HACS (Recommended)

1. Open **HACS**.
2. Open **Integrations**.
3. Click **+ Explore & Download Repositories**.
4. Search for **Intervals.icu**.
5. Install the integration.
6. Restart Home Assistant.

---

## Manual Installation

1. Download the latest release.
2. Copy the `custom_components/ha_intervals_icu` folder into your Home Assistant `custom_components` directory.
3. Restart Home Assistant.
4. Add the integration.

---

# Configuration

Go to:

**Settings → Devices & Services → Add Integration**

Enter:

- Athlete ID
- API Key

The API Key can be generated from:

**Intervals.icu → Settings → Developer Settings**
